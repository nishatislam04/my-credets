1. backend - Bun
2. frontend - tanstack router
3. ui library - shadcn
4. databases - postgresql
5. devOps - docker, docker-compose(?)
6. email - resend
7. validation - zod
8. backend hosting platform - Render
9. frontend hosting platform - (( not sure yet ))


# requirements

create a web application where we can securely host our crednentials.
create new credentials, update existing credentials, delete unnecessary!
bulk export credentials with email
full auth and authorization for each routes 
also make sure to add media uploading support
make the credentials input very flexible. as they dont follow a strict structure
