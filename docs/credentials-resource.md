# this is the actual page to describe our credentials-resource information

## the create form

1. title - text input
2. types - see below
3. short description -text-area (will be hidden by default behind a button)
4. long description -text-area (will be hidden by default behind a button)
5. thumbnail image (single image)
6. images (single or multiple images)
7. the DATA input (see below)
8. notes -text-area
9. tags -special tag component


### the data input part from the create form

we will have some presets (buttons) like this -

1. button label - single value. generate a single text input
2. button label - key pair. generate 2 text inputs with key and value
3. button label - information. generate a text-area input.

### types input field from the create form

when we click the form input for the first time,
all the existing types will be listed.
user can eithe choose one of them or
create a new one type by simply typing it in input box
